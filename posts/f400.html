Hexo 部署前的环境准备

Hexo官方文档提示需要安装 nodejs 和 Git，其中 nodejs 的版本不能低于 8.10，推荐 10.0 及以上。其实，这里的 nodejs 中还包括了 npm 这个包管理器，用来下载安装 Hexo 的。如果下载的 nodejs 中没有 npm 的话还需要安装 npm，当然这是后续之需，我们先来下载 nodejs 吧！

安装 NodeJs：

  • NodeJs 官网：https://nodejs.org
  • NodeJs 中文官网： https://nodejs.org/zh-cn/

通过apt或者apt-get包管理器安装

用node -v查看版本号信息，验证是否安装成功。

nodejs 里面自带 npm，不需要单独安装 npm。如果没有 npm 的话，可以执行命令 sudo apt install npm 来安装一下。
